# Telegram Channel AI Digest Bot

Бот для Telegram с двумя режимами работы:
1. **Ручная публикация** — создание текстовых/медиа постов, планирование, черновики.
2. **AI-выжимка** — автоматический сбор новостей из Telegram-каналов и генерация дайджеста через OpenAI.

---

## Быстрый старт

```bash
# 1. Установи зависимости
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt

# 2. Создай .env из примера и заполни
cp .env.example .env

# 3. Запусти
python main.py
```

Для работы выжимки достаточно заполнить `BOT_TOKEN`, `ADMIN_ID`, `CHANNEL_ID` и `OPENAI_API_KEY`.  
Telethon — необязателен (см. ниже).

---

## Режимы чтения каналов

### Web-режим (по умолчанию, без регистрации)

Бот парсит публичные страницы `https://t.me/s/<channel>`. Не требует никаких Telegram API-ключей.

**Ограничения:** работает только с **публичными** каналами; страница показывает ~20 последних постов.

### Telethon-режим (расширенный)

Читает каналы через Telegram API как обычный пользователь. Даёт доступ к приватным каналам и большей истории.

Активируется автоматически, если заполнены `TELETHON_API_ID` и `TELETHON_API_HASH` в `.env`.  
При первом запуске бот попросит ввести код из SMS — сессия сохраняется в `data/telethon_session`.

## Настройка .env

| Переменная | Обязательная | Описание |
|---|---|---|
| `BOT_TOKEN` | ✅ | Токен от @BotFather |
| `ADMIN_ID` | ✅ | Telegram ID администратора |
| `CHANNEL_ID` | ✅ | ID или @username канала для публикации |
| `OPENAI_API_KEY` | Для выжимки | Ключ OpenAI или совместимого API |
| `TELETHON_API_ID` | Нет | С my.telegram.org (только для приватных каналов) |
| `TELETHON_API_HASH` | Нет | С my.telegram.org |
| `TELETHON_PHONE` | Нет | +79001234567 |
| `OPENAI_BASE_URL` | Нет | По умолчанию `https://api.openai.com/v1` |
| `OPENAI_MODEL` | Нет | По умолчанию `gpt-4o-mini` |
| `DIGEST_HOURS` | Нет | Глубина выборки новостей (по умолч. 24 ч.) |
| `DIGEST_MAX_POSTS` | Нет | Макс. постов с канала (по умолч. 20) |

## Команды бота

### Ручная публикация
| Команда / Кнопка | Действие |
|---|---|
| `/start` | Главное меню |
| `/newpost` / `✏️ Новый пост` | Создать пост (текст / фото / видео) |
| `/drafts` / `📋 Черновики` | Список черновиков |
| `/scheduled` / `🕐 Запланированные` | Запланированные посты |
| `/history` / `📜 История` | История публикаций |
| `/cancel` | Отменить текущее действие |
| `/help` | Справка |

### AI-выжимка
| Команда / Кнопка | Действие |
|---|---|
| `/digest` / `📰 Выжимка` | Собрать новости и сгенерировать выжимку |
| `/sources` / `📡 Источники` | Показать список источников |
| `/addsource @channel` | Добавить канал-источник |
| `/removesource @channel` | Удалить источник |
| `/settings` | Текущие настройки и активный режим чтения |

### Кнопки черновика выжимки
| Кнопка | Действие |
|---|---|
| ✅ Опубликовать | Отправить в канал; новости помечаются как обработанные |
| 🔁 Переделать | Перегенерировать выжимку из тех же постов |
| 📝 Изменить текст | Вручную отредактировать текст перед публикацией |
| ❌ Отмена | Удалить черновик |

## Получение Telethon API (необязательно)

Нужно только если хочешь читать **приватные** каналы:

1. Зайди на [my.telegram.org](https://my.telegram.org) и авторизуйся.
2. Перейди в **API development tools**.
3. Создай приложение, скопируй `api_id` и `api_hash`.

## Частые вопросы

**Бот не публикует в канал?**  
Убедись, что бот добавлен администратором канала с правом публикации. Логи — в `logs/bot.log`.

**Канал не добавляется в web-режиме?**  
Канал должен быть публичным. Проверь вручную: открой `https://t.me/s/<username>` в браузере — должны отображаться посты.

**Telethon просит код повторно?**  
Удали файл `data/telethon_session*` и перезапусти — бот пройдёт аутентификацию заново.

**Хочу другой AI-провайдер (OpenRouter, Mistral и т.д.)?**  
Укажи `OPENAI_BASE_URL` и `OPENAI_MODEL` под их значения — `openai` пакет поддерживает любой OpenAI-совместимый API.
